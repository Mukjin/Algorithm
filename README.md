# Algorithm

알고리즘 문제 해결 및 코딩 테스트 준비를 위한 리포지토리입니다.

## 📚 개요

이 저장소는 다양한 온라인 저지 플랫폼에서 제공하는 알고리즘 문제들을 풀이하고 정리한 것입니다.

- **백준 (BOJ)**: 프로그래밍 기초부터 고급 알고리즘까지 다루는 한국 최대규모의 온라인 저지
- **프로그래머스**: 코딩 테스트 준비를 위한 문제 풀이
- **SW Expert Academy**: 삼성 소프트웨어 검증 과정(SWEA) 문제 풀이

## 💻 사용 언어

- **Java**: 97.3% - 주요 풀이 언어
- **Python**: 2.7% - 보조 풀이 언어

```
Algorithm/
```
